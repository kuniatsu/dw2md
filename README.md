# DeepWiki One-Tap Markdown Exporter 🚀

DeepWikiの情報を「ワンタップ」で収集し、NotebookLMへ橋渡しするための専用ツールです。

## 📖 プロジェクト概要
このツールは、社内ツール **DeepWiki** に蓄積された膨大なドキュメントを、AI解析ツール **Google NotebookLM** で即座に活用できるように開発されました。

### 解決する課題
- **認証の壁:** ログイン必須の環境であるため、外部ツールではデータが取得できない。
- **作業コスト:** 数十ページあるドキュメントを1枚ずつコピー＆ペーストするのは現実的ではない。
- **構造の複雑さ:** プロジェクトごとにページ数が異なり、手動での管理が煩雑。

本ツールはブラウザ拡張機能として動作し、**「ログインしているあなたの権限」**で安全に、かつ**「一括」**で全ページのテキストをMarkdown形式で抽出します。

---

## ✨ 主な機能

- **サイドバー自動解析:** ルート画面の左サイドバーにあるリンクをすべて自動で検出します。
- **バックグラウンド一括取得:** ページを一枚ずつ開く必要はありません。裏側で全ページにアクセスし、内容を読み取ります。
- **Markdown自動成形:** HTMLタグを解析し、見出し・リスト・テーブルなどをNotebookLMが理解しやすいMarkdown形式に変換します。
- **シングルファイル出力:** バラバラのページを一つのファイルに結合して出力するため、NotebookLMへのアップロードは一度で済みます。

## 🛠 インストールガイド（初回のみ）

非エンジニアの方でも以下の手順で3分で設定完了します。

1. **資材の準備:**
   - 配布された `deepwiki-exporter.zip` を解凍し、フォルダ（`deepwiki-exporter`）をデスクトップなどの分かりやすい場所に置きます。
2. **拡張機能画面へアクセス:**
   - ChromeブラウザのURLバーに `chrome://extensions/` と入力して開きます。
3. **開発者モードの有効化:**
   - 画面右上の **「デベロッパー モード」** スイッチを **ON** にします。
4. **ツールの読み込み:**
   - 左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンを押し、手順1のフォルダを選択します。

## 🚀 使い方

1. **DeepWikiへログイン:**
   - 通常通りDeepWikiにアクセスし、対象プロジェクトのトップ画面（ルート画面）を表示します。
2. **ボタンをクリック:**
   - 画面右下に現れる **「🚀 一括エクスポート」** ボタンを押します。
3. **完了を待つ:**
   - 画面上に進捗が表示されます（例: 5/30 ページ取得中...）。
4. **NotebookLMへアップロード:**
   - ダウンロードされた `.md` ファイルを、[NotebookLM](https://notebooklm.google.com/) のソースに追加してください。

## ❓ 困ったときは（FAQ）

**Q. ボタンが表示されません**
- ページを再読み込み（F5キー）してください。また、ルート画面（サイドバーに全ページが並んでいる状態）であることを確認してください。

**Q. 途中で止まってしまいます**
- ネットワーク環境やWikiの応答速度により時間がかかる場合があります。数十ページある場合は30秒ほどお待ちください。

**Q. 特定のページだけ中身が空になります**
- そのページに閲覧権限があるか確認してください。ブラウザでそのページが開ける状態であれば、ツールでも取得可能です。

## ⚙️ 技術仕様（管理者向け）
- **対象ドメイン:** `app.devin.ai`
- **変換エンジン:** `Turndown.js` (HTML to Markdown)
- **抽出ロジック:** - サイドバーセレクター: `#app > ... > ul` 内のリンクを走査
  - 本文セレクター: `#app > ... > div` のコンテンツを抽出
- **セキュリティ:** ブラウザのCookieセッションを利用した `fetch` 通信。外部サーバーへのデータ転送は一切行いません。
