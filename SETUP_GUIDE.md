# DeepWiki Markdown Exporter - 初心者導入マニュアル

**対象読者:** パソコンの基本操作ができる方（エンジニア経験は不要です）
**所要時間:** 約3分
**必要なもの:** Google Chrome ブラウザ

---

## このツールでできること

DeepWikiに蓄積されたドキュメント（数十ページ）を、**ボタン1つ**で1つのファイルにまとめてダウンロードできます。
ダウンロードしたファイルは Google NotebookLM にそのままアップロードして、AIに質問できるようになります。

**Before（これまで）:** ページを1つずつ開いてコピー＆ペースト → 何十分もかかる
**After（このツール）:** ボタンを1回押すだけ → 数十秒で完了

---

## STEP 1: ファイルを準備する

### 1-1. フォルダを入手する

配布された `deepwiki-exporter` フォルダを用意してください。
中に以下の3ファイルが入っていることを確認します。

```
deepwiki-exporter/
  |- manifest.json    ... 拡張機能の設定ファイル
  |- turndown.js      ... 変換エンジン
  |- content.js       ... メインの処理
```

### 1-2. フォルダを置く場所

デスクトップなど、**分かりやすい場所**に置いてください。

> **注意:** このフォルダは削除しないでください。削除すると拡張機能が動かなくなります。

---

## STEP 2: Chromeに拡張機能を登録する

### 2-1. 拡張機能の管理画面を開く

Chrome ブラウザのアドレスバー（URLを入力する場所）に、以下を入力して Enter キーを押します。

```
chrome://extensions/
```

### 2-2. デベロッパーモードを ON にする

画面の**右上**にある「デベロッパー モード」のスイッチを探してください。
**OFF（灰色）** になっている場合は、クリックして **ON（青色）** に切り替えます。

```
  [ デベロッパー モード ] ←ここをクリック
  OFF → ON に変更
```

### 2-3. フォルダを読み込む

画面の**左上**に「パッケージ化されていない拡張機能を読み込む」というボタンが表示されます。

1. そのボタンをクリック
2. フォルダ選択ダイアログが開きます
3. STEP 1 で置いた `deepwiki-exporter` **フォルダを選択**
4. 「フォルダーの選択」をクリック

### 2-4. 登録を確認する

拡張機能の一覧に **「DeepWiki One-Tap Markdown Exporter」** が表示されていればOKです。
スイッチが **ON（青色）** になっていることを確認してください。

---

## STEP 3: エクスポートを実行する

### 3-1. DeepWiki にログインする

普段どおり DeepWiki（`app.devin.ai`）にアクセスし、ログインしてください。

### 3-2. 対象プロジェクトのルート画面を開く

左側のサイドバーに、プロジェクト内の**全ページ名が一覧表示**されている画面を表示してください。
これが「ルート画面」です。

```
 ┌──────────────────────────────────────────────┐
 │  サイドバー        │   メインコンテンツ       │
 │                    │                          │
 │  > gymflow-api     │   System Overview        │
 │  > vetclinic-core  │   ...                    │
 │  > stock-warehouse │                          │
 │  > platform-nexus  │                          │
 │    ...             │                          │
 └──────────────────────────────────────────────┘
              ↑ この一覧が見えている状態
```

### 3-3. エクスポートボタンを押す

画面の**右下**にピンク色のボタンが表示されます。

```
                          ┌──────────────────────┐
                          │ 🚀 一括エクスポート    │
                          └──────────────────────┘
```

このボタンをクリックしてください。

### 3-4. 完了を待つ

画面中央に進捗表示が出ます。全ページの取得が完了するまでお待ちください。

```
  ┌─────────────────────────────┐
  │      エクスポート中...        │
  │                             │
  │  3/10 ページ取得中...        │
  │  vetclinic-core             │
  │                             │
  │  ████████░░░░░░░░░  30%     │
  └─────────────────────────────┘
```

**ページ数の目安:**
- 5ページ程度 → 約5秒
- 20ページ程度 → 約15秒
- 50ページ程度 → 約30秒

### 3-5. ファイルがダウンロードされる

完了すると、`.md` ファイルが自動的にダウンロードされます。
ファイル名はプロジェクト名になります（例: `aurora-labs_pet-care_vetclinic-core.md`）。

ダウンロードフォルダを確認してください。

---

## STEP 4: NotebookLM にアップロードする

1. [Google NotebookLM](https://notebooklm.google.com/) を開く
2. 「新しいノートブック」を作成する（または既存のものを開く）
3. 「ソースを追加」をクリック
4. 「ファイルをアップロード」を選択
5. STEP 3 でダウンロードした `.md` ファイルを選択

これで NotebookLM にドキュメント全体が取り込まれ、AIに質問できるようになります。

## よくある質問（トラブルシューティング）

### Q. ピンク色のボタンが表示されません

**原因 1: ページが違う**
サイドバーに全ページの一覧が表示されている「ルート画面」でのみボタンが出ます。
個別ページを直接開いている場合は、サイドバーの「Wiki」リンクをクリックしてルート画面に戻ってください。

**原因 2: 読み込みが完了していない**
ページの読み込み中はボタンが出ません。数秒待ってから `F5` キーでページをリロードしてください。

**原因 3: 拡張機能が無効になっている**
`chrome://extensions/` を開いて、「DeepWiki One-Tap Markdown Exporter」の
スイッチが ON になっていることを確認してください。

---

### Q. エクスポート中に画面が固まったように見えます

正常です。バックグラウンドでページを1つずつ取得しているため、ページ数が多い場合は
時間がかかります。進捗バーが少しずつ進んでいれば問題ありません。
**ブラウザを閉じずにお待ちください。**

---

### Q. ダウンロードされたファイルの中身が一部空です

そのページへの**閲覧権限がない**可能性があります。
ブラウザで該当ページを直接開いて、内容が表示されるか確認してください。
表示できないページは、ツールでも取得できません。

---

### Q. 「エクスポート完了（○件のエラーあり）」と表示されました

一部のページの取得に失敗しました。以下を試してください。

1. ネットワーク接続を確認する
2. ページを `F5` でリロードして、再度ボタンを押す
3. それでも改善しない場合は、管理者にお問い合わせください

---

### Q. 拡張機能を削除したい

1. `chrome://extensions/` を開く
2. 「DeepWiki One-Tap Markdown Exporter」の「削除」ボタンをクリック

---

### Q. 拡張機能を更新したい（新しいバージョンを受け取った場合）

1. 配布された新しい `deepwiki-exporter` フォルダで古いフォルダを**上書き**する
2. `chrome://extensions/` を開く
3. 「DeepWiki One-Tap Markdown Exporter」の中にある更新ボタン（丸い矢印アイコン）をクリック

---

## セキュリティについて

- このツールは**あなたのブラウザ内だけ**で動作します
- 外部サーバーへのデータ送信は**一切行いません**
- DeepWikiへのアクセスには、あなたが**ログイン済みのセッション**（Cookie）を使用します
- ソースコードはすべて公開されており、透明性が確保されています
